## 🕡 Como baixar o projeto
```bash

   # clonar repositório
   $ git clone https://github.com/EnzoVizotto/dart

   # entrar no diretório
   $ cd projeto-final

   # instalar as dependencias
   $ yarn install

   #iniciar o projeto
   $ yarn start

```
